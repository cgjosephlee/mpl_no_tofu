# No Tofu
Eat all the tofus in your matplotlib plot!

# Quick guide
## Installation
```
pip install git+https://github.com/cgjosephlee/mpl_no_tofu.git
```

## Load fonts
```
import matplotlib.pyplot as plt
from mpl_no_tofu import load_fonts
load_fonts()  # installs "NotoSansTC"
# your plots
```
